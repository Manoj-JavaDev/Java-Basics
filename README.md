# Number Guessing Game (Java)

## Description
This is a simple console-based Number Guessing Game written in Java.
The program generates a random number within a user-defined range,
and the user must guess the number within limited attempts.

## Features
- User-defined number range
- User-defined maximum attempts
- Hints for higher or lower guesses
- Random number generation

## Tools Used
- Java JDK 25
- Eclipse IDE

## How to Run
1. Open the project in Eclipse
2. Run the GuessNumber.java file
3. Follow the console instructions

## Learning Outcome
- Java basics
- Loops and conditional statements
- Random number generation
- User input using Scanner
