# Java Basics

## Description
This repository contains basic Java practice programs.
These programs are created to understand core Java concepts.

## Tools & Technologies
- Java JDK 25
- Eclipse IDE

## Topics Covered
- Variables and Data Types
- Loops
- Arrays
- Conditional Statements

## How to Run
1. Install JDK 25
2. Open the project in Eclipse
3. Run the Java files individually
